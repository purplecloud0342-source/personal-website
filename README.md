# Personal Website

https://purplecloud0342-source.github.io/personal-website/

个人作品集网站，展示 AI 项目和视频作品的双语单页应用。中/英切换，响应式布局，滚动动画。

## 技术栈

React 19 + TypeScript + Vite · Tailwind CSS v4 · GSAP ScrollTrigger · Framer Motion · Lucide Icons

## 开发

```bash
npm install
npm run dev       # http://localhost:3000
npm run build     # 构建到 dist/
npm run preview   # 预览构建产物
```
